# 🛡️ RAKSHA - Smart AI-Based Surveillance System

A smart, AI-powered **home surveillance system** that enhances your security using **real-time face recognition**, **motion & object detection**, and **instant alerts** via **email** and **push notifications** — all built with **Firebase**, **OpenCV**, and modern web technologies.

---

## 🚀 Features

- 🔐 **Live Surveillance Dashboard** with secure login  
- 🧠 **Real-time Face Recognition** with whitelist management  
- 🎯 **Motion & Object Detection** to identify suspicious activity or harmful items  
- 📸 **Activity Logging** with timestamps and captured images, securely stored in Firebase  
- 📧 **Email Alerts**  
  - Named alerts for recognized (whitelisted) individuals  
  - Warnings for unknown faces or unusual motion  
- 🔊 **Voice Announcements** for unrecognized persons  
- 📲 **Push Notifications** using Firebase Cloud Messaging (FCM)  
- ☁️ **Firebase Integration** for authentication, real-time database, and cloud storage  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask, OpenCV)  
- **Database & Authentication:** Firebase (Firestore, Firebase Auth, Firebase Storage, FCM)  
- **AI/ML:** OpenCV, TensorFlow / scikit-learn (for anomaly and object detection)
